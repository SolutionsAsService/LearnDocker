Managing Data and Networking with Docker

In this tutorial, you will learn about Dockerâ€™s networking modes and how to persist data using volumes. Key topics include:

  • Understanding Docker network modes (bridge, host, overlay)
  • Creating custom networks to link containers
  • Using volumes for persistent data storage

Creating a Custom Docker Network


# Create a new bridge network
docker network create my-network

# Run a container and connect it to the network
docker run -d --name my-container --network my-network nginx:latest

Using Docker Volumes

Create and use a Docker volume for persistent storage:


# Create a Docker volume
docker volume create my-volume

# Run a container using the volume
docker run -d --name data-container -v my-volume:/data nginx:latest

These practices help ensure reliable container communication and data persistence in your Docker environment.
