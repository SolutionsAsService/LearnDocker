Creating a Dockerfile and Building Images

In this tutorial, you will learn how to create a Dockerfile to containerize your application. Topics include:

  • Writing a Dockerfile with best practices
  • Building an image using docker build
  • Tagging and running the resulting image

Sample Dockerfile

Create a file named Dockerfile with the following content:


# Use an official Node.js image as the base
FROM node:14-alpine

# Set the working directory
WORKDIR /usr/src/app

# Copy package files and install dependencies
COPY package*.json ./
RUN npm install

# Copy the rest of the application code
COPY . .

# Expose the port your app runs on
EXPOSE 3000

# Start the application
CMD [ "node", "server.js" ]

Building and Running the Image


# Build the Docker image and tag it as "my-node-app"
docker build -t my-node-app .

# Run the container mapping port 3000
docker run -d -p 3000:3000 my-node-app

By following these steps, you'll be able to containerize your Node.js application using Docker.
